PC 웹 로그인 화면을 디자인해줘.

전체 레이아웃:
- 화면 전체는 밝은 배경
- 로그인 카드를 화면 정중앙(수직 + 수평 중앙)에 배치
- 카드 크기는 가로 400~480px, 세로 자동

구성 요소 (위에서 아래 순서):
1. 로고 이미지 영역 — 카드 상단 중앙에 직사각형 이미지 영역
   (실제 로고 이미지가 들어갈 자리)

2. 아이디 입력 필드 — 흰색 배경의 넓은 둥근 직사각형, 카드 가로 꽉 채움

3. 비밀번호 입력 필드 — 흰색 배경의 넓은 둥근 직사각형, 카드 가로 꽉 채움

4. 하단에 버튼 두 개를 나란히 배치:
   - 왼쪽: "로그인" — 둥근 직사각형, 어두운 파란색 (#1a3a6b)
   - 오른쪽: "회원가입" — 둥근 직사각형, 밝은 보라색 (#7c5cbf)

스타일:
- 전체 배경: 밝은 회색 또는 흰색
- 로그인 카드: 다크 네이비 (#0a1628), 큰 border-radius, 그림자 효과
- 입력 필드: 흰색 배경, 어두운 테두리, 둥근 모서리
- 버튼 텍스트: 흰색
- 전체적으로 깔끔하고 심플한 스타일
