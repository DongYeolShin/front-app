## 1. 목적
- 대문페이지 작성

## 2. 구성
- 화면을 1:2 비율로 나눔
- 1 에는 텍스트
- 2 서울 오늘 날씨 표시 


## 3. UI 디자인

### 3.1 좌측 구성
- 텍스트를 삽입
    - 오늘의 날씨에 대해서 설명 
    - '''
       **오늘의 날씨입니다** 
       오늘의 날씨는
        최저 10도 / 최고 20도 
        풍속 3m/s 이고 풍향은 북서풍입니다.
      '''

### 3.2 우측 구성
- 날씨 API를 이용한 서울 날씨 표시
- 순서는 날씨 아이콘, 날씨, 최저/최저 온도, 풍속, 풍향

## 3.3 디자인
- 배경색은 옅은 회색

- 텍스트는 배경색 대비 잘보이는 색으로 설정
- 날씨쪽 배경은 따로 없음 


## API 정보
- end-point : https://api.openweathermap.org/data/2.5/weather
- 전송방식 : get
- 전송데이터:
    - units=metric
    - q = seoul
- 아이콘 :
    - https://openweathermap.org/img/wn/${icon}@2x.png
    - icon은 데이터에서 추출하여 입력하면 됨.
